This code requires any version of the following libraries:
opencv | pip install opencv-python
numpy | pip install numpy
time (it can be removed from code)| pip install time-python 

Inside the code, there is a section "input".
You can configure the size of the square filter (changing "a" variable),
configure type of filter (Low Pass/High Pass) (changing "c" variable),
change VideoCapture camera input.

To quit program when running, press "q".
